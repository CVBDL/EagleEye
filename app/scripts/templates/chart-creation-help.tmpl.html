Details about chart options

What's major axis data types?
The major axis is the axis along the natural orientation of the chart. For line, area, column, combo, stepped area charts, this is the horizontal axis. For a bar chart it is the vertical one. In line, area, bar, column (and combo charts containing only such series), you can control the type of the major axis:
1. For a discrete axis, set the data column type to string.
2. For a continuous axis, set the data column type to one of: number, date or datetime.

What's stacked chart?
A stacked bar chart is a bar chart that places related values atop one another. If there are any negative values, they are stacked in reverse order below the chart's axis baseline.
